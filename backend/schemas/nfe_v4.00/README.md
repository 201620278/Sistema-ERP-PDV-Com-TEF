# Schemas NF-e 4.00

## Arquivos necessários

Coloque aqui os schemas oficiais da NF-e versão 4.00:

1. **enviNFe_v4.00.xsd** - Schema do lote de envio
2. **leiauteNFe_v4.00.xsd** - Leiaute da NF-e
3. **tiposBasico_v4.00.xsd** - Tipos básicos
4. **xmldsig-core-schema_v1.01.xsd** - Schema de assinatura digital

## Download

Baixe os schemas oficiais em:
- Site da Fazenda: https://www.nfe.fazenda.gov.br/portal/listaConteudo.aspx?tipoConteudo=/fwLvLGoMmtk=
- ou Pacote de Schemas NF-e 4.00 (XSD) - versão atual

## Observação

Sem esses arquivos, a validação do XML falhará com erro "Schema não encontrado".
A aplicação funcionará, mas não será feita a validação XSD antes do envio.
